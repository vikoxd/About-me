Setting repository

1.Masuk ke terminal, lalu ketik “su” dan masukkan password root nya.
2.Ketik “ping google.com” lalu enter.Tekan “Ctrl+c” untuk menghentikan tulisan.




3.Kemudian ketik “apt-get update”.Setelah itu maka akan muncul seperti dibawah ini, kemudian ketik “nano /etc/apt/sources.list” lalu enter




4.Maka akan muncul tampilan menu seperti ini, kemudian cari data utama di link https://www.linuxsec.org/2020/11/daftar-repository-lokal-indonesia.html.




5.Lalu ketik semua yang ada di data utama dan jika tampilannya sudah seperti dibawah, tekan Ctrl+X lalu Y lalu enter.




6.Lalu ketik “apt update” lalu enter. Jika mendapat pertanyaan “Continue” ketik enter.




7.Lalu ketik “apt upgrade” lalu enter,dan akan muncul tampilan seperti ini
Jika ada pertanyaan Y/n jawab y lalu enter




8.Jika selesai akan muncul tampilan seperti ini.




9.Ketik perintah “apt-get install ssh” kemudian enter,jika ada pertanyaan “Continue” ketik enter.



10. Ketik perintah “nano /etc/ssh/ssh_config” kemudian enter. Jika sudah, maka tampilannya seperti ini, kemudian cari “# Port 22” lalu hapus tanda pagar nya. Kemudian jika sudah tekan Crl+X lalu Y lalu enter
11. Lalu ketik perintah “systemctl restart ssh” dan enter. Kemudian ketik perintah “systemctl status ssh” Lalu tekan enter,dan akan muncul tampilan seperti ini.





12. Kemudian ketik “ip -c a” lalu enter.




13. Lalu ketik “nano /etc/network/interfaces”. Kemudian enter,lalu akan ada tampilan seperti ini, Kemudian ketik “auto enp0s3” sampai ke bawah (diisi sesuai dengan IP Laptop kalian), Jika sudah tekan Crl+X lalu Y lalu enter.




14. Kemudian, cek ping jaringan di terminal dengan cara ketik perintah “ping (Ip Kalian).” Kemudian enter.
